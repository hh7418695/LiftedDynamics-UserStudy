## 1. Executive Summary

### 1.1 Study Goals

This psychophysics study investigates how humans perceive object softness through haptic feedback using a TouchX haptic device. Specifically, we aim to:

1. **Quantify discrimination thresholds** for stretching stiffness (axial deformation) and bending stiffness (angular deformation)
2. **Compare perceptual sensitivity** between stretching and bending modalities
3. **Generate psychometric curves** relating physical stiffness to perceptual judgments
4. **Identify just-noticeable differences (JNDs)** for each stiffness dimension

### 1.2 Expected Duration

- **Per participant:** 45-60 minutes (including consent, training, breaks, and debriefing)
- **Pilot testing:** 2-3 participants (Week 1-2)
- **Main study:** 20-30 participants (Week 3-8)
- **Total study duration:** 8-10 weeks (including data analysis)

### 1.3 Number of Participants

**Target sample size:** N = 20-30 participants

**Rationale:** Based on power analysis for psychometric curve fitting:
- Effect size (Cohen's d) = 0.8 (large effect expected for stiffness discrimination)
- Power (1-β) = 0.80
- Alpha (α) = 0.05
- Minimum N = 15 per condition (stretching vs. bending)
- Recommended N = 20-30 to account for potential exclusions and ensure robust curve fitting

### 1.4 Key Outcomes

**Primary outcomes:**
1. Discrimination accuracy (% correct) for each object pair
2. Psychometric curves (proportion "stiffer" vs. physical stiffness)
3. Point of subjective equality (PSE) and just-noticeable difference (JND)

**Secondary outcomes:**
1. Response times
2. Confidence ratings (if implemented)
3. Subjective reports of difficulty
