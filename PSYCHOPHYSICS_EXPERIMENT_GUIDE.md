## 1. 实验概述

### 1.1 研究目标

本实验旨在研究人类通过触觉反馈感知虚拟物体柔软度的心理物理特性，具体包括：

- **主要目标**：
  - 测量拉伸刚度（stretching stiffness）对柔软度感知的影响
  - 测量弯曲刚度（bending stiffness）对柔软度感知的影响
  - 确定刚度辨别阈值（Just Noticeable Difference, JND）
  - 建立物理刚度与主观柔软度的心理物理函数

- **次要目标**：
  - 分析交互力学特征与感知的关系
  - 探索触觉错觉现象
  - 研究个体差异

### 1.2 系统架构

本实验使用双组件实时触觉渲染系统：

```
┌─────────────────┐         UDP          ┌──────────────────┐
│  C++ UDP Client │ ◄─────────────────► │  Python Dynamics │
│  (TouchX Device)│   Position/Force    │     Engine       │
│   Windows PC    │      ~1000 Hz       │   (JAX/LNN)      │
└─────────────────┘                      └──────────────────┘
```

- **C++ 客户端**: 接口 TouchX 触觉设备，1kHz 采样率
- **Python 服务器**: 基于拉格朗日神经网络的物理引擎
- **通信**: UDP 协议，实时双向数据传输

## 2. 实验设计

### 2.1 实验范式

推荐采用以下三种实验范式：

#### 实验 A：两区间强迫选择（2AFC - Two-Alternative Forced Choice）

**目的**: 测量刚度辨别阈值（JND）

**设计**：
- **自变量**: 刚度差异（ΔStiffness）
- **因变量**: "更硬"判断的准确率
- **方法**: 恒定刺激法（Method of Constant Stimuli）

**试验流程**：
```
1. 呈现标准刺激（Standard）- 参与者探索对象A（30秒）
2. 短暂间隔（5秒）
3. 呈现比较刺激（Comparison）- 参与者探索对象B（30秒）
4. 强迫选择："哪个对象更硬？" → A 或 B
5. 记录响应和反应时间
```
